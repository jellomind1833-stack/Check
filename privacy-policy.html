Lumen Mail Privacy Policy

Developer: WHstellar

Last updated: March 15, 2026

Overview

Lumen Mail is a Chrome extension that helps users clean Gmail Promotions and Social messages. This Privacy Policy explains what data the extension processes, how it is used, and what is not collected or shared.

What Lumen Mail Processes

  • Gmail page content required to identify Promotions and Social message rows.
  • Basic page state required to select messages and trigger Gmail delete actions.
  • User preferences saved through Chrome extension storage, such as delete limit, language, and UI accent settings.

What Lumen Mail Does Not Collect

  • No personal data is sent to any developer server.
  • No analytics, advertising trackers, or third-party SDKs are used.
  • No browsing data outside https://mail.google.com/* is accessed.
  • No payment, health, location, or authentication data is collected.

How Data Is Used

Data is processed only to provide the extension’s single purpose: cleaning Gmail Promotions and Social messages at the user’s request. Stored settings are used only to remember the user’s chosen configuration for future runs.

Storage and Retention

Lumen Mail uses Chrome extension storage on the user’s device and Chrome account sync storage for lightweight settings. The extension does not maintain its own remote database and does not retain Gmail message content outside the active browser session.

Data Sharing

Lumen Mail does not sell, transfer, or share user data with third parties, except when required by law. User data is not used for advertising, creditworthiness, lending decisions, or unrelated profiling.

Google API Services and Limited Use

If the extension ever uses information received from Google services, it will comply with the Chrome Web Store Limited Use requirements. The current version operates on the Gmail web interface and does not transmit Gmail content to the developer.

User Control

  • Users may stop cleanup at any time from the extension popup.
  • Users may change or clear saved settings from the extension options page.
  • Users may disable or remove the extension at any time from Chrome.

Contact

For privacy questions about Lumen Mail, contact the developer through the Chrome Web Store listing or the published support channel for WHstellar.

This policy should match the data disclosures and permissions shown in the Chrome Web Store listing for Lumen Mail.
